Assignment 2

In this assignment, you will create a Java program containing three classes named AdminGrants, ProgramGrant, and Charity. These classes will be used to compute a charity's rating (for an imaginary Charity).

  1. The ProgramGrant class represents the rating and subjective impact factor received for a program grant. A program grant is any expense that directly relates to carrying out the charity's mission. The formula used to calculate the program grant rating is rating = (number of people impacted by expense / cost)*100. It must:
    • have a private instance variable that keeps track of the rating
    • have a private instance variable that keeps track of am impact grade (as a string)
    • have a constructor which takes three arguments, a double, an int and a String, which are the amount of the grant, number of people impacted, and subjective impact factor respectively
    • have ``get'' functions to get the rating and subjective impact factor
    • ensure that the cost of the grant and number of people impacted by the grant is non-negative (if one is negative, set it equal to 0)
    • ensure that the impact grade is either A, B, C, D, or F. If a value other than those is used, a warning message should be printed to the console that an invalid impact factor was entered, and the value ``A'' should be used.
    • ensure that the rating is not greater than 100 (if it is greater than 100, set it equal to 100)


  2. The AdminGrants class represents the combined amount of money spent on administrative salaries. It must:
    • have a private instance variable that keeps track of the total (sum) money spent on administrative costs
    • have a private instance variable that keeps track of the number of paid employees (it's possible this is 0)
    • have a function that allows you to add an administrative expense (this function needs to ensure the expense is non-negative)
    • have a function that allows you to set the number of paid employees (this function needs to ensure the number of paid employees is non-negative)
    • have a function that returns the average administrative cost per paid employee
    • have a function that returns the total amount of administrative grants


  3. The Charity class represents a charity and is used to for entering grants and computing and printing the rating and grade of the charity. For this assignment, we suppose each charity has two program grants: short term grants and research grants. It must:
    • have a private instance variable that keeps track of the Charity's name
    • have a private instance variable that keeps track of the number of paid employees
    • have s private instance variable of type ProgramGrant that keeps track of the charity's short term program grants
    • have s private instance variable of type ProgramGrant that keeps track of the charity's research program grants
    • have a private instance variable of type AdminGrants that keeps track of the charity's administrative grants
    • have a private instance variable that keeps track of the Charity's rating (which is a double)
    • have a private instance variable that keeps track of the Charity's grade (a string)
    • have a private instance variable that keeps track of the Charity's total expenses
    • have a constructor which takes the total expenses of the charity (a double), the number of salaried employees and the name of the charity as arguments
    • have the following member functions (implemented):
      		public void addResearchGrant(double amount, int numPeopleImapcted, String impactGrade)
		public void addShortGrant(double amount, int numPeopleImapcted, String impactGrade)
		public void addAdminGrant(double amount)
		public void printReport()
    • printReport() should calculate and output the Charity's rating as follows:
      		(short term grant rating * 40%) +
		(research grant rating * 30%) +
		(administrative grant rating * 30%)
	Based on the charity's rating, a letter grade should be computed as:
		F (<60), D (≥60, <70), C (≥70, <80), B (≥80, <90), A (≥90)
      The administrative grant rating is given by (1 - (administrative grants total/ total expenses))*100. Follow the formula strictly (i.e. a score of 79.88% is a C and doesn't round to 80% and a B).


  4. To make it more interesting, let's say that we are harsh critics. The final letter grade awarded to the charity cannot be any higher than the lowest impact grade awarded for any program grant. Thus, if the letter grade calculated above is higher than the lowest subjective impact factor, the new final grade should be the lowest subjective impact factor grade. In addition, if the average salary of the paid employees is higher than $100,000.00, the charity should be awarded a letter grade of F.


  5. The Charity's name with their final letter grade and rating (between 0 and 100, rounded to exactly two decimal places) must be output to the console. The output line must begin with "Final rating for" and then include the charity's name, letter grade, and rating.
