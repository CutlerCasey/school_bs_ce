Assignment 7

  1. For this assignment, you will be creating an HTML page with JavaScript functions and classes which mimic the functionality of your assignments 2 and 3. That is, you will be creating Charity, AdminGrants, and ProgramGrant classes which can be used to compute the letter grade and rating for a Charity. You must write your page "by hand", meaning that you are not allowed to use an HTML editor or application which writes HTML for you, and all code should be contained within a single file, which is the only file you will submit.


  2. Create a file named FirstLast.html (where "First" is replaced with your first name and "Last" with your last name). Create a title for your page (using the "title" tag) and a main heading (using the "h1" tag) with the text "First Last's Assignment 7" (again substituting your name).


  3. Other than the heading, your page should only display a form which requests the Charity's data. The form should contain 11 text input fields and one "Submit" button. The input fields should request:
    1. the charity's name
    2. total amount of expenses
    3. a comma separated list of each administrative grant cost
    4. the amount for the research grant
    5. number of people impacted by the research grant
    6. the impact factor (a letter grade) of the research grant
    7. the amount for the short term grant
    8. the number of people impacted by the short term grant
    9. the impact factor (a lettr grade) of the short term grant
    There should be text next to each input field specifying which data to enter into it. Note: For easier testing, you can set your input text fields to have default test values, and obviously these will be changed by the grader when grading.


  4. Set your form so that when the submit button is pressed, a JavaScript function named "calculateGrade" is called. Hint: Use the "onsubmit" attribute of the form as follows: onsubmit="return calculateGrade()". Also, make sure that your "calculateGrade" function always returns false. This will prevent the browser from "submitting" the form and then reloading the page after the submit button is pressed. Make your "calculateGrade" function do something basic like call alert and display data from the form (the name entered, etc.) to ensure that you've got everything working up to this point. Don't continue until you do.


  5. Create JavaScript classes Charity, AdminGrants, and ProgramGrant which have the same attributes and methods as those classes did in assignment 2. They should be structured the sam. Further, all attributes of each class should be "private" and the methods should all be "priveleged" (see http://javascript.crockford.com/private.html for details).


  6. Modify "calculateGrade" so that it gathers all of the necessary data from the form and uses it to create a Charity object and set all necessary score and grade values. Have it call alert with the string returned by the "getGrade" method of the Charity class. Make sure this is working before continuing (including the correct arithmetic for computing final letter grades, etc.).


  7. Make one more set of modifications to "calculateGrade" so that instead of displaying the charity's grade in an alert popup, it prints that information to the HTML page. The heading should remain visible on the page, but the entire form should now be hidden. Thus, when the page is loaded and the charity's information is added into the form and then the submit button is pushed, the form should disappear and the correct grade for the charity should be displayed instead (in the format specified for assignment 2, i.e. "Final rating for: Charity_Name X (yy%)"). Don't worry about rounding the percentage score to two decimal places or about the spacing on your page, meaning that it doesn't matter where the output information about the charity is located, as long as the form is not visible and the heading is.
